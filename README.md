---

## 📁 Structure

```
DSA-Java/
├── Arrays/                       # Array problems & patterns
├── Binary search/                # Binary search problems (Lec 36–37)
├── Sorting/                      # Sorting algorithms (Lec 35)
└── Striver's SDE sheet/
    └── Arrays/                   # LeetCode problems from SDE Sheet
        ├── BuySellStock.java     # Best Time to Buy & Sell Stock
        ├── pascalsTriangle.java  # Pascal's Triangle
        └── setMatrixZero.java    # Set Matrix Zeroes
```
